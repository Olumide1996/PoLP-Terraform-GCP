3. Networking Aspects

DNS and IP Addresses

  • DNS:
    The domain name (or subdomain) is translated to an IP address via DNS. This resolution process ensures users browsers can locate the server hosting your site.
  • IP Addresses:
    The hosting provider (Netlify) manages the servers public IP address. Clients connect to the servers IP address when they visit your website.

Protocols

  • HTTP/HTTPS:
    The website uses HTTP for data transfer. HTTPS is enforced to secure data in transit. HTTPS uses TLS (Transport Layer Security) to encrypt the data between the client and the server.
  • Port Usage:
    Standard ports are used (80 for HTTP and 443 for HTTPS).

where to find etc/hosts in windoes

In Windows, the equivalent of the /etc/hosts file on Unix-based systems is located at:

makefile
C:\Windows\System32\drivers\etc\hosts

How to Access and Edit the Hosts File:

  1. Open File Explorer:

    • Navigate to C:\Windows\System32\drivers\etc.

  2. Viewing the File:

    • By default, you might not see any file extensions. Look for a file named hosts (without an extension).

  3. Editing the Hosts File:

    • Important: You need administrative privileges to edit this file.
    • Method 1: Using Notepad as Administrator
      • Click on the Start menu, type Notepad, then right-click on Notepad and choose "Run as administrator."
      • In Notepad, click File > Open and navigate to C:\Windows\System32\drivers\etc.
      • In the "Open" dialog, change the file type from "Text Documents (.txt)" to "All Files (.*)" to see the hosts file.
      • Select the hosts file, make your edits, then save.
    • Method 2: Using Other Text Editors
      • If you prefer another text editor like Visual Studio Code, make sure to run it as an administrator (right-click the application and select "Run as administrator"), then open the file from C:\Windows\System32\drivers\etc.

Note:

  • Always create a backup of the original hosts file before making changes.
  • After editing, you might need to flush the DNS cache for changes to take effect. Open Command Prompt as administrator and run:
    bash
    ipconfig /flushdns

By following these steps, you'll be able to locate, access, and safely edit the hosts file on your Windows machine.
